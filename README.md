# ROT13 + LSB Steganography (Python)

Aplikasi ini dibuat untuk memenuhi tugas UAS mata kuliah **Kriptografi** dengan topik kelompok **ROT13 + Steganografi LSB**.  
Program ini melakukan proses:

1. **Enkripsi ROT13**
2. **Embedding pesan (ciphertext) ke dalam gambar menggunakan LSB**
3. **Ekstraksi ciphertext dari gambar stego**
4. **Dekripsi kembali menggunakan ROT13**
5. **Validasi apakah gambar memiliki pesan tersembunyi**
